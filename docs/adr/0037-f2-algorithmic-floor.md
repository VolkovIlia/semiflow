# ADR-0037 — F2 algorithmic floor disclosure (Krylov vs Taylor)

**Status**: Proposed 2026-05-19 for v0.13.0.

**Context**: Iter-3 bench F2 (1D heat time-dep coef, Magnus class) measured `TruncatedExp4thDiffusionChernoff` 65× slower than kiops-magnus-gl4 on identical accuracy targets. Wave B (ADR-0019 Amendment 2, ADR-0034 Amendment 1) closes ~3.4-4.5× via SIMD + storage refactor + sample bypass, leaving **15-20× residual gap**. Researcher analysis (Gaudreault et al. 2018, KIOPS, J. Comput. Phys. 372:236-255, arXiv 1804.05126) confirms this residual is **algorithmic**, not implementation-bound: KIOPS uses adaptive Krylov subspace with 5-15 Arnoldi vectors × 1 mat-vec each + adaptive substepping ≈ 15-30 A-applications per step; RC TruncatedExp4 K=4 = 4 stencil applications + 5 cubic-Hermite samples per node, architecturally fixed regardless of τ. At low Krylov rank on smooth-coefficient problems, KIOPS is fundamentally more efficient per accuracy unit.

**Decision**: Disclose the 15-20× residual as DELIBERATE architectural trade-off in `CHANGELOG.md` and `BENCHMARKS-STATUS.md`, not a bug to be fixed. RC retains H-MEM Pareto dominance (kiops uses 125× more memory at F2 per iter-3 §5.1 table). Switching to Krylov-Magnus would either (a) violate ROADMAP.md:354 "Chernoff approach is explicit by design" — Krylov-Arnoldi requires implicit-style Hessenberg basis assembly, or (b) duplicate the kiops crate's design, providing no novel contribution. We accept the speed-memory Pareto position. Future work: if a user case emerges where speed-Pareto matters more than memory-Pareto on F2-class problems, evaluate Krylov-Magnus as a separate crate (per ADR-0012 precedent for FFT-spectral).

**Consequences**: F2 verdict in iter-4 bench (post-v0.13.0) will be INCONCLUSIVE or REJECT_NULL on wall-time while RC retains H-MEM SUPPORT (125× less memory than kiops-magnus-gl4 per iter-3 table). Positioning: RC is memory-first Magnus class; kiops is speed-first. No fix is owed; documentation owed. Citation: Gaudreault et al. "KIOPS: A fast adaptive Krylov subspace solver for exponential integrators" J. Comput. Phys. 372 (2018) 236-255, DOI 10.1016/j.jcp.2018.06.026.
