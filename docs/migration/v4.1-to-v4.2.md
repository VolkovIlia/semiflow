# Migration Guide: v4.1 → v4.2 (Path ε QuinticHermite spatial upgrade, AMENDMENT 1)

**ADR-0089 / Path ε Wave I, AMENDMENT 1 (Option A modified)** — QuinticHermite (O(dx⁶))
spatial sampling is available on all kernels via opt-in builder methods. Defaults have been
chosen to preserve v4.1 byte-equality for ζ⁴ while delivering the QuinticHermite benefit
for ζ⁶ unconditionally.

## What changed (AMENDMENT 1 vs Wave I Wave-I engineer default-ON)

| Kernel | v4.1 default | v4.2 default | Opt-in |
|--------|-------------|-------------|--------|
| `Diffusion4thChernoff` (K5) | CubicHermite | **unchanged** (CubicHermite) | `.with_quintic_sampling()` |
| `Diffusion4thZeta4Chernoff` (ζ⁴) | CubicHermite | **unchanged** (CubicHermite) | `.with_quintic_sampling()` |
| `Diffusion6thZeta6Chernoff` (ζ⁶) | CubicHermite | **QuinticHermite** (direct K5 wiring) | always on |

**ζ⁴ default unchanged from v4.1**: QuinticHermite default-ON (Wave-I engineer change)
regressed G_zeta4_const_a Richardson ratio from 3.58 → 3.226 (below 3.5 gate).
ADR-0089 AMENDMENT 1 reverts ζ⁴ to CubicHermite default, restoring gate passage.

**ζ⁶ directly wires K5 to Quintic**: ζ⁶'s n={1,2} measurement is insensitive to
Quintic's pre-asymptotic shift (benefit only), lifting G_zeta6 from 3.67 → 3.868.
This is unconditional in `Diffusion6thZeta6Chernoff::new()`.

## No breaking changes for ζ⁴ users

`Diffusion4thZeta4Chernoff::new()` is unchanged from v4.1: CubicHermite default,
works for any `F: SemiflowFloat`. No f64-only guard. No `Unsupported` error.

## Opt-in to QuinticHermite on ζ⁴ (power user)

```rust
use semiflow_core::{Diffusion4thChernoff, Diffusion4thZeta4Chernoff, Grid1D};

let grid = Grid1D::new(-10.0_f64, 10.0, 512).unwrap();
let k5 = Diffusion4thChernoff::new(|_| 1.0, |_| 0.0, |_| 0.0, 1.0, grid);

// v4.2 ζ⁴ default: CubicHermite (same as v4.1)
let zeta4 = Diffusion4thZeta4Chernoff::new(k5.clone(), Some(1.0)).unwrap();

// Power-user opt-in to QuinticHermite on ζ⁴ (ADR-0089 §"Path ε"):
let zeta4_quintic = zeta4.with_quintic_sampling();

// Downgrade back to CubicHermite:
let zeta4_cubic = zeta4_quintic.without_quintic_sampling();
```

## ζ⁶ automatically uses QuinticHermite inner K5

```rust
use semiflow_core::{Diffusion4thChernoff, Diffusion4thZeta4Chernoff,
    Diffusion6thZeta6Chernoff, Grid1D};

let grid = Grid1D::new(-10.0_f64, 10.0, 512).unwrap();
let k5 = Diffusion4thChernoff::new(|_| 1.0, |_| 0.0, |_| 0.0, 1.0, grid);
let zeta4 = Diffusion4thZeta4Chernoff::new(k5, Some(1.0)).unwrap();

// zeta4.inner.quintic_sampling == false (CubicHermite default)
let zeta6 = Diffusion6thZeta6Chernoff::new(zeta4, Some(1.0)).unwrap();
// zeta6.inner.inner.quintic_sampling == true (direct K5 Quintic wiring in new())
// G_zeta6_const_a ratio: 3.868 (passes gate 3.8)
```

## K5 base QuinticHermite opt-in (unchanged from ADR-0089 Wave I)

```rust
// Opt in to QuinticHermite on K5 directly:
let k5_quintic = k5.with_quintic_sampling();

// Revert to CubicHermite:
let k5_cubic = k5_quintic.with_cubic_sampling();
```

## Deprecation preview: `new_cubic()`

`Diffusion4thZeta4Chernoff::new_cubic()` is deprecated in v4.2 (same as `new()` now).
Will be removed at v4.3.0 per ADR-0035 §9. Migrate to `new()`.

## ζ⁸ Wave II — DEFERRED to v4.3+

`Diffusion8thZeta8Chernoff` (Wave II) was evaluated in Phase B of v4.2.0.
The nested-Richardson ratio log₂ measured **3.067**, falling below the
architectural floor of 4. ADR-0088 AMENDMENT 2 (445c893) records this result
and formally defers ζ⁸ to v4.3+. No ζ⁸ type ships in v4.2.0; ζ⁶ remains
the top rung of the Richardson ladder at this release.

Users who anticipated ζ⁸ availability at v4.2 should use `Diffusion6thZeta6Chernoff`
as the highest-order Richardson kernel. The `.with_quintic_sampling()` infrastructure
(Path ε, ADR-0089) is in place for any future ζ⁸ attempt.

## v5.0 global default promotion

At v5.0 BREAKING window, the plan is to promote QuinticHermite to the
`Grid1D::new` default for all kernels (ADR-0089 §"Future"). Until then, only
ζ⁶ uses Quintic by default (via direct K5 wiring), and all other kernels
retain CubicHermite as the default.
